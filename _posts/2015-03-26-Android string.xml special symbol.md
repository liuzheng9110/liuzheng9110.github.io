最近项目中有需要在string.xml 中显示特殊符号，如@ © 等，使用ASCII码进行显示： 

常用
```
@号	    &#064;   
%百分号	&#037;   
:号	    &#058;   
空格	  &#160;   
©	      &#169;   
®	      &#174;   
```
